# Talent Hub 🚀

Talent Hub is a full-stack job board web application that connects recruiters and job seekers on a single platform. Recruiters can post and manage job opportunities, while candidates can explore jobs and apply by uploading their resumes.

The project is designed with a modern and responsive user interface to provide a smooth user experience across all devices.

## Features

- User Authentication & Authorization
- Recruiter Job Posting System
- Candidate Job Application
- Resume Upload Functionality
- Email Notifications
- Search and Filter Jobs
- Responsive Design
- Secure Backend APIs

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- JavaScript

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Project Purpose

The main goal of Talent Hub is to simplify the recruitment process by creating a platform where companies can hire talented candidates and job seekers can easily apply for jobs.

This project helped in learning:
- Full Stack Development
- REST APIs
- Authentication
- File Upload Handling
- Deployment
